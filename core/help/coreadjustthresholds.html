Description

There are three possible types:


highisgood: the higher the better

lowisgood: the lower the better

optimalvalue: there is an optimal value for this metric.
Choose the tresholds HIGHISGOOD

When the user chooses highisgood, the next fields are:

Minimum green value. This becomes greenThreshold

Minimum amber value (lower than greenThreshold). This becomes amberThreshold.

LOWISGOOD

When the user chooses lowisgood, the next questions are:

Maximum green value. This becomes greenTreshold

Maximum amber value (higher than greenTreshold). This becomes amberTreshold

OPTIMALVALUE

When the user chooses optimalvalue, the next questions are:

Optimal value. This becomes optimalvalue

Minimum green value (lower than optimalvalue). This becomes greenTresholdDown Minimum amber value

lower than greenTresholdDown). This becomes amberTresholdDown Maximum green value (higher than optimalvalue). This becomes greenTresholdUp

Maximum amber value (higher than greenTresholdUp). This becomes amberTresholdUp
