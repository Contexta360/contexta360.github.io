Word cloud filter

Apart from the Metric management menu, there is one additional feature for Boost that is not in Core.
It should become possible to select a filter in the word cloud on the Status pages. The filter can look identical to the filter on the Interactions page in the Agent menu.
The goal of this functionality is to show significant terms for conversations that perform good or bad on certain metrics. This greatly enhances the possibilities to find interesting patterns for metrics.

Endpoint: analytics/wordcloud
Body:
foregroundFilter: the converted filter for the metric. This is already implemented for the Interactions filter on the Agent page.
backgroundFilter: just the selected time and channel.
