Description
This page enables you to create your own use cases. From here you will assign metrics to your use cases.
The logical flow is:
Create use case, create or assign an existing metric, create or assign an existing query.
A maximum of 4 use cases can be displayed on the status menu. Use cases and metrics can be toggled on/off, depending on display choices.
