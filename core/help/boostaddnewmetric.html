Description

Create a new metric. Once created add a query to the metric and assign it to a use case.

New metrics are set to 'off' by default. In order to turn 'on' you may need to switch another one 'off' as only 4 metrics can be switched on at a time. Once switched on and assigned to a use case, the data will be available in the relevant application screens.

A metric consists of the following fields:
shortName: a short name of the metric. Max 15 characters. name: the name of the metric. element: textfield

Description: a longer description of the field. element: text area source: the source of the metric, see following section tresholds: the metric tresholds, see following section:

T create a metric, first choose the source type. A metric can have two types of possible sources:
Query Metadata field

The user can choose a query as the source of the metric. When type "query" is selected, a list of queries should appear which he can then select. For instance with a filter dropdown.
